# 产品愿景

一个简单的四象限画布工具——用户可以自由定义横轴与纵轴的名字、颜色、刻度范围，然后在画布上添加文字、卡片、图标等元素并拖拽编辑。

## 典型场景

将横坐标定义为"重要程度"、纵坐标定义为"紧急程度"，用户即可在画布上对事务进行优先级安排（经典艾森豪威尔矩阵）。同理可用于波士顿矩阵、影响/努力矩阵、风险评估矩阵等。
