# 使用指南

## 界面概览

应用由四部分组成：

- **顶栏**：工具切换、当前工具指示、导入/导出/保存按钮
- **左侧栏**：坐标轴配置、图层列表、快速模板
- **画布**：四象限网格，放置和拖拽元素
- **右侧栏**：选中元素的属性面板

## 工具栏

| 工具 | 快捷键 | 说明 |
|------|--------|------|
| 选择 | — | 点击选中元素，拖拽移动 |
| 文字 | — | 双击画布创建文字元素 |
| 矩形 | — | 双击画布创建卡片元素 |
| 圆形 | — | 双击画布创建图标元素 |
| 卡片 | — | 双击画布创建卡片元素 |
| 图标 | — | 双击画布创建图标元素 |
| 连线 | — | 预留功能 |

## 快捷键

| 按键 | 功能 |
|------|------|
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Delete` / `Backspace` | 删除选中元素（焦点不在输入框时） |

## 坐标轴配置

在左侧栏"坐标轴"选项卡中可以：

- 修改轴标题和端点标签（实时同步到画布）
- 选择轴颜色
- 设置刻度范围
- 切换轴线显示/隐藏

## 添加元素

- **双击画布**任意位置即可添加一个新元素，元素类型取决于当前选中工具
- 新元素默认名称为"新元素"，可在右侧属性面板中编辑

## 编辑元素

选中元素后，右侧面板可编辑：

- 文本内容和 Emoji
- 精确位置（X / Y）
- 字体大小（增减按钮）
- 颜色（色板选择）
- 不透明度（滑块）
- 元素类型（卡片/标签/图标/文字）
- 图层顺序（上移/下移）
- 删除元素

## 图层管理

切换到"图层"选项卡查看所有元素列表：

- 点击元素行选中对应画布元素
- 点击眼睛图标切换元素可见性

## 快速模板

在"模板"选项卡中，点击预设模板可一键修改轴配置：

- 时间管理矩阵（艾森豪威尔矩阵）
- 波士顿矩阵
- 影响/努力矩阵
- 风险评估矩阵

## 保存与导出

- **自动保存**：每次状态变更后 400ms 自动写入浏览器 localStorage
- **手动保存**：点击顶栏"保存"按钮立即写入
- **导出 JSON**：点击"导出"，下载包含完整状态的 `.json` 文件
- **导入 JSON**：点击"导入"，从 `.json` 文件恢复状态

## 缩放

底部状态栏右侧提供缩放控件：

- `−` / `+` 按钮以 10% 步进调整
- "适应"按钮重置为 100%

## 已知限制

- 连线工具为预留功能，暂未实现
- 画布尺寸固定为 680×620px，缩放通过 CSS transform 实现
- 触摸设备暂未专门适配
- 导出为 PNG 图片功能尚未实现
